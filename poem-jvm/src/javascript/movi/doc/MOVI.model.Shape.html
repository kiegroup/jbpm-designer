Class MOVI.model.Shape - extends YAHOO.util.Element

Known Subclasses:
MOVI.model.Edge MOVI.model.Canvas MOVI.model.Node
An abstract base class providing a wrapper object for all model shapes (including the canvas, nodes, and edges).

Constructor

MOVI.model.Shape ( jsonObj , stencilset , parent , prefix , attr )
Parameters:
jsonObj <Object> The JSON object from which the new shape is created
stencilset <Object> The stencilset for lookup of the shape's stencil
parent <Shape> The shapes's parent shape
prefix <String> The element's ID prefix (unique per modelviewer)
attr <Object> (optional) A key map of the shape element's initial attributes

Properties

parentShape - object

The shapes parent shape
